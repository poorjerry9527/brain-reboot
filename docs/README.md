# 📖 日常即永恒 · 我的现象学日记本

> 「人当诗意地栖居」——荷尔德林  
> 而诗意，不过把最普通的二十四小时，过成可以反复咀嚼的琥珀。

---

## 为什么记录？

| 哲学回声 | 我的转译 |
|---|---|
| **苏格拉底**「未经省察的人生不值得过」 | 若不给日子打一束反光，它便永远沉入暗河。 |
| **亨利·柏格森**「时间是绵延的质地」 | 当文字落在纸面，绵延被剪成可回溯的薄片。 |
| **海德格尔**「此在总是先行向死」 | 把每一天当成「有限性」的切片，才会郑重其事。 |
| **加缪**「必须想象西西弗斯幸福」 | 重复洗衣、通勤、买菜——我在推石上山的途中，仍能对夕阳吹口哨。 |

## 阅读提示（给未来的我，也给你）

1. **别急着「总结意义」**  
   让意义像青苔，慢慢爬满石阶。

2. **允许重复**  
   尼采的「永恒轮回」提醒我：若同一天重演万次，你是否仍愿这样度过？答案藏在字缝。

3. **保留矛盾**  
   今天的我与昨天的我辩驳，正如黑格尔的正反合——日记是思维的沙盘，不是判决文书。

4. **庆祝琐碎**  
   本雅明迷恋「拱廊街」的尘埃，我迷恋菜市场落日、地铁玻璃里的倒影——微小即宏大。
